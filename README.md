# SkreenApp

SkreenApp es una aplicación de escritorio para Linux que permite extender o transmitir la pantalla del equipo a un dispositivo Android mediante una conexión USB. Además, aprovecha la pantalla táctil del dispositivo móvil para interactuar con el sistema desde el ordenador.

Actualmente, el proyecto se encuentra en desarrollo y se están implementando nuevas funcionalidades, como la creación de monitores virtuales para ampliar el espacio de trabajo.

---

## Características

- Transmisión de pantalla de Linux a dispositivos Android.
- Conexión mediante cable USB.
- Soporte para interacción táctil desde el dispositivo móvil.
- Bajo consumo de ancho de banda al no requerir conexión Wi-Fi.
- Próximamente: creación de monitores virtuales.

---

## Instalación

### Debian / Ubuntu (`.deb`)

```bash
sudo apt install ./skreenapp_0.1.0_amd64.deb
```

### Fedora / RPM (`.rpm`)

```bash
sudo dnf install ./skreenapp-0.1.0-1.fc43.x86_64.rpm
```

---

## Desinstalación

### Debian / Ubuntu (`.deb`)

```bash
sudo apt remove skreenapp
```

### Fedora / RPM (`.rpm`)

```bash
sudo dnf remove skreenapp
```
